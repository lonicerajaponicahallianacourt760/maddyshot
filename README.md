## 🎨 Using the tool

The interface uses a simple layout to help you create images fast. 

### Writing your code
Click the central text box. Type or paste your code here. The software automatically detects your programming language and adds color to the syntax. This makes your code easier to read.

### Customizing the look
Use the side panel to change how your image appears.
* **Theme:** Pick custom color schemes to match your style.
* **Padding:** Adjust the space around your code block.
* **Background:** Select a solid color or a gradient for your image background.

### Adding Persian support
If you use Persian text in your comments or strings, the tool handles it automatically. It correctly aligns text from right to left to ensure the Persian characters display without errors. Select a font from the settings menu that supports Persian glyphs for the best results.

## 🖼️ Saving your image
When the image looks right, click the Export button. The program asks you to choose a folder on your computer. Select a name for your file and save it as a PNG or JPG image. You can now attach this file to emails or post it to your preferred platforms.

* Keep your code snippets short. Large blocks of code look better on smaller images.
* Check your font settings if Persian characters do not show up as expected.
* Use the snapshot feature to copy the image to your clipboard instead of saving a file.
